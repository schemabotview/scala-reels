# scala-reels

Content (narration `tts/` + generated `audio/`) for the **Scala** topic in
[graphl-mobile](../graphl-mobile), the technical reels PWA. The app fetches the
`audio/<stem>.wav` at runtime via raw GitHub; it is not bundled into the app.

See **[CLAUDE.md](./CLAUDE.md)** for structure, the authoring/audio pipeline, and
the scene index.
